Overview:
Laravel 12 web app for organizing trips, activities, bookings, and budgets with interactive maps and charts.

Collaborators:
Solo project

Introduction

Itinerary Planner lets authenticated users create trips, plan daily activities, manage bookings and participants, and track budgets in one place. Locations are set through a draggable Leaflet map and spending insights are visualized with Chart.js.

Details

  • CRUD management for itineraries, activities, bookings, and group members.
  • Budget entries grouped by category with Excel and PDF export options.
  • Dashboard with search and date filters plus dark/light theme support.
  • Responsive Tailwind CSS and Alpine.js front end built with Vite.
