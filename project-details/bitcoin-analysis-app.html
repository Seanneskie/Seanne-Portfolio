Overview:
A web-based Bitcoin data analysis and prediction tool built using Django and Python data science libraries. This project analyzes historical Bitcoin prices and provides users with statistical summaries, visualizations, and linear regression-based predictions.

Collaborators:
Individual Project

Introduction

Bitcoin’s hallmark volatility makes it a compelling subject for time-series and predictive analytics. This study set out to:

  1. quantify central-tendency & dispersion metrics for 2019-2021 prices,
  2. visualise relationships among price variables,
  3. build per-column linear-regression models and cross-check them with Weka outputs,
  4. translate the analysis into an interactive web demo for non-technical users.

Findings

  • Perfect positive correlation among Open, High, Low, Close, and Weighted Price columns confirms their synchronous movement in the market.
  • Linear-regression models trained per price column achieved consistent error rates and mirrored Weka’s outputs, boosting confidence in methodology.
  • Descriptive stats underline Bitcoin’s extreme volatility; high standard deviations and wide value ranges signal substantial short-term risk / reward potential.
  • Forward projections indicated potential upward drift for major price indicators, offering directional insight for traders while reinforcing the caveat that crypto markets remain highly unpredictable.
