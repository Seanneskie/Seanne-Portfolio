Overview:
The CNSM Website was a semester-long Advanced Database Project undertaken during SY 2023-2024 by second-year BSIT students. Its goal was to build a full-featured web portal for the College of Natural Sciences and Mathematics (CNSM)—centralizing announcements, course materials, laboratory schedules, and faculty–student interactions—while showcasing practical application of a MERN (MongoDB, Express, React, Node) architecture. Despite solid early momentum, the project was paused near the semester’s end due to time constraints and competing coursework.

Collaborators:
Second-year BSIT students Database Major

Introduction

The project stemmed from our Advanced Database course requirement to marry sound data modeling with modern web practice. We chose MERN to gain end-to-end JavaScript proficiency and version-controlled everything in a shared GitHub repository, enforcing pull-request reviews for peer learning. Key planned modules included:

  1. Dynamic CMS — admins post news, events, and lab notices.
  2. Student Dashboard — personalized timetable, subject resources, and grade visibility.
  3. Faculty Tools — laboratory booking, file uploads, and class analytics.
  4. Role-Based Access — JWT-secured routes differentiating admin, faculty, and student privileges.

By mid-semester we had scaffolding, auth flow, and base CRUD for announcements deployed on a Heroku test instance. However, finals and overlapping projects curtailed further feature completion. The repo now stands as a learn-as-you-go artifact illustrating MERN best-practice patterns, which can be revived for future cohorts.
