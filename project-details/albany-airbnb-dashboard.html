Overview:
An interactive dashboard built with HTML, CSS, and JavaScript to explore Airbnb listings in Albany, NY. It visualizes prices, availability, and guest sentiment through charts and maps.

Collaborators:
Solo project for ITD112 - Data Analysis and Visualization

Introduction

Using data from InsideAirbnb, this project presents local rental trends and review sentiment in an accessible way. Users can inspect average prices, availability, and neighborhood statistics.

Details

Features include clustered maps, summary cards, radar charts, and availability calendars that let users compare listings and explore pricing differences across Albany neighborhoods.

Objectives

  • Develop an interactive dashboard using the Albany Airbnb dataset.
  • Visualize listing distribution with heatmaps and clustered maps.
  • Display key averages through summary cards.
  • Compare single listings against overall metrics.
  • Analyze sentiment, pricing, and neighborhood trends.

Dashboard Features

  • Clustered Map with Popups: explore listings without clutter.
  • Interactive Icons & Legends: quickly identify property types.
  • Summary Cards: show average price, bedrooms, bathrooms and beds.
  • Radar Chart and Tables: compare a selected property to dataset averages.
  • Gantt & Line Charts: reveal availability over time.
  • Boxplots by Neighborhood: understand ward price differences.

Insights and Findings

  • Sentiment Analysis: guest reviews are overwhelmingly positive.
  • Price Distribution: most listings fall between $51 and $150.
  • Neighborhood Trends: some wards command higher prices than others.
  • Event Impact: spikes in bookings occur during local events.

Usage Tips

  • Use the clustered map to zoom in on specific neighborhoods.
  • Hover on the radar chart to compare a property with the average.
  • Check the calendar views to spot popular booking dates.

Dataset Source

Data is sourced from Inside Airbnb and processed with Python's Pandas library before visualization.

How to Run

  1. Clone this repository.
  2. Open albany-airbnb-dashboard/index.html in your browser.

Interact with the map, cards, and charts to explore the data.
