# One Oasis Davao — photos

Drop photos in this folder, then reference them from
`src/content/travels/one-oasis-davao.md` using site-root paths:

- `cover.jpg`          → `/static/travels/one-oasis-davao/cover.jpg`
- `01.jpg`, `02.jpg`…  → `/static/travels/one-oasis-davao/01.jpg`

**Filename hygiene:** lowercase, no spaces, hyphens not underscores
(`pool-view.jpg` not `Pool View.JPG`).

**Recommended sizes:** cover ~1600px wide, gallery ~1200px wide.
Re-export JPEGs at quality 80 for a good size/quality tradeoff.

> `01.jpg` is a poster frame extracted from a video clip (this stop's
> source had only `.MOV` files). Replace with a still photo when available.
