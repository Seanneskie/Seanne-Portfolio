# Starbucks SM Lanang — photos

Drop photos in this folder, then reference them from
`src/content/travels/starbucks-sm-lanang.md` using site-root paths:

- `cover.jpg`          → `/static/travels/starbucks-sm-lanang/cover.jpg`
- `01.jpg`, `02.jpg`…  → `/static/travels/starbucks-sm-lanang/01.jpg`

**Filename hygiene:** lowercase, no spaces, hyphens not underscores
(`latte-art.jpg` not `Latte Art.JPG`).

**Recommended sizes:** cover ~1600px wide, gallery ~1200px wide.
Re-export JPEGs at quality 80 for a good size/quality tradeoff.
