Order & Inventory Management API

Built a lightweight Order & Inventory Management API with ASP.NET Core and PostgreSQL.

  • Product CRUD with stock validation and idempotent order placement.
  • Transactional updates and inventory audit logging.
  • Uses EF Core migrations, FluentValidation, RFC 7807 ProblemDetails, Swagger, and Serilog.
  • Optimistic concurrency via Postgres xmin prevents oversells.
  • Integration tests run against Dockerized PostgreSQL. Designed for Azure deployment.
