Order & Inventory Management API

Lightweight ASP.NET Core and PostgreSQL service providing product CRUD, stock validation, and idempotent order placement.

Transactional updates with an inventory audit log ensure consistency while Postgres xmin optimistic concurrency prevents oversells.

EF Core migrations, FluentValidation, RFC 7807 ProblemDetails, Swagger, and Serilog support robust development. Integration tests run against a Dockerized Postgres instance and the API is designed for Azure deployment.
